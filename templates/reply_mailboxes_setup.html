📥 Connect Contact Mailboxes

Authorize the Gmail / Outlook accounts you'll use as test "contacts". After sending an email, the Get reply button on each recipient row will fire a real AI-generated reply from that contact's mailbox back to the sender.

Each account you connect here becomes a reachable test contact. The refresh token is stored securely in your Neon database so the app can send a reply on that contact's behalf when you click Get reply. You can disconnect any contact at any time.

How this works

  1. Click Connect contact Gmail or Connect contact Outlook above.
  2. Sign in with the contact account and grant send & read mail permissions.
  3. You'll return to this page once the OAuth flow completes; the new mailbox appears in the list above.
  4. Repeat for every test contact you want to be able to "reply" from.
  5. Go back to the main app and use Get reply on a recipient row to trigger a real inbound reply.
