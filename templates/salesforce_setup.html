Quick steps

  1. Salesforce Setup → App Manager → New Connected App.
  2. Tick Enable OAuth Settings. Paste the Callback URL above.
  3. Selected OAuth Scopes: Manage user data via APIs (api) and Perform requests at any time (refresh_token, offline_access).
  4. Save, wait ~10 minutes for Salesforce to propagate, then copy the Consumer Key and Consumer Secret here.
  5. If the org uses My Domain or is a sandbox, change the Login URL accordingly.
